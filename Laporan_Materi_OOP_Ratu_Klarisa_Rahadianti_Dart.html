LAPORAN MATERI OBJECT ORIENTED PROGRAMMING (OOP) - DART

1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Objek dibuat dari sebuah kelas (class) yang berisi atribut (data) dan method (fungsi).

  • Class dan Object
  • Encapsulation
  • Constructor
  • Method

2. Contoh Class dan Object (Tanpa Constructor)

class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

3. Constructor

class Car {
  String? model;
  String? color;

  Car(this.model, this.color);

  void start() {
    print('Car started!');
  }
}

4. Validasi Data (Class Mahasiswa)

class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus berada dalam rentang 0.0 - 4.0');
    }
  }
}
▶ Buka di DartPad

5. Encapsulation

class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}
▶ Buka di DartPad

6. Constructor dengan Named Parameter

class Produk {
  final String id;
  final String nama;
  final double harga;
  final int stok;
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = 'Umum',
  });
}
▶ Buka di DartPad

7. Tugas Bonus (Soal No.3 Halaman 15)

class Hewan {
  String nama;

  Hewan(this.nama);

  void bersuara() {
    print('$nama mengeluarkan suara.');
  }
}

void main() {
  Hewan kucing = Hewan('Kucing');
  Hewan anjing = Hewan('Anjing');

  kucing.bersuara();
  anjing.bersuara();
}
▶ Buka di DartPad

8. Kesimpulan

OOP mempermudah pengelolaan program besar karena kode lebih terstruktur, mudah dikembangkan, dan lebih aman melalui konsep enkapsulasi dan constructor. Paradigma ini sangat cocok digunakan dalam pembuatan aplikasi menggunakan Dart dan Flutter.
